# Agumino

This is the web-based React version of the app preview.

## Run

```bash
npm install
npm run dev
```

Then open the local URL shown by Vite.

## Main Files

- `src/App.jsx` contains the React components and app behavior.
- `src/styles.css` contains the visual design.
- `public/agumino-logo.png` is the logo asset.
